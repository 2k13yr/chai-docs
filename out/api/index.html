The Chai API is broken down by style or task.

  • The Expect / Should API covers the BDD assertion styles.
  • The Assert API covers the the TDD assertion style.

If you are interested in developing plugins, then the Plugin API will be of use.

If you are interested in knowning what your browser is capable of, the Online Test Suite covers compatibility for the most recent version.
